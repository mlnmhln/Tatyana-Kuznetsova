# Лендинг Татьяны Кузнецовой

Одностраничный адаптивный сайт на HTML, CSS и JavaScript.

## Локальный просмотр

Запустите в папке проекта любой статический сервер, например:

```bash
python3 -m http.server 8080
```

После этого откройте `http://localhost:8080`.

## Подключение формы

Форма подготовлена для Cloudflare Pages Functions и отправки заявок в Telegram. В настройках проекта задайте секреты:

- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_CHAT_ID`

В локальном режиме форма выполняет безопасную демо-отправку и не передает персональные данные.
